Function signature:
```python
def wmrl_m6b_hierarchical_model_subscale(
    participant_data_stacked: dict,
    covariate_matrix: jnp.ndarray | None = None,
    covariate_names: list[str] | None = None,
    num_stimuli: int = 6,
    num_actions: int = 3,
    q_init: float = 0.5,
    wm_init: float = 1.0 / 3.0,
) -> None:
```

Where `covariate_matrix` has shape `(n_participants, n_covariates)` and `covariate_names` is e.g. `["lec_total", "iesr_total", "iesr_intr_resid", "iesr_avd_resid", "iesr_hyp_resid"]`.

**Implementation pattern:**

ALL 8 parameters are sampled manually (NOT via `sample_bounded_param`) so that L2 shifts can be applied uniformly to each one. The `sample_bounded_param` helper is bypassed entirely in this function because it does not support multi-covariate L2 shifts. Instead, each parameter follows the manual pattern: `{param}_mu_pr`, `{param}_sigma_pr`, `{param}_z` sites, with L2 shifts added on the unconstrained scale before the phi_approx transform.

```python
# For each parameter that gets L2 regression:
param_names = ["alpha_pos", "alpha_neg", "phi", "rho", "capacity", "epsilon",
               "kappa_total", "kappa_share"]

# Sample L2 coefficients if covariate_matrix provided
betas = {}
if covariate_matrix is not None:
    for pname in param_names:
        for cov_name in covariate_names:
            site_name = f"beta_{cov_name}_{pname}"
            betas[(pname, cov_name)] = numpyro.sample(site_name, dist.Normal(0.0, 1.0))

# For each parameter:
sampled = {}
for pname in param_names:
    defaults = PARAM_PRIOR_DEFAULTS[pname]
    mu_pr = numpyro.sample(f"{pname}_mu_pr", dist.Normal(defaults["mu_prior_loc"], 1.0))
    sigma_pr = numpyro.sample(f"{pname}_sigma_pr", dist.HalfNormal(0.2))
    z = numpyro.sample(f"{pname}_z", dist.Normal(0, 1).expand([n_participants]))
    
    # Unconstrained individual-level
    theta_unc = mu_pr + sigma_pr * z
    
    # Add L2 shifts if covariates provided
    if covariate_matrix is not None:
        for j, cov_name in enumerate(covariate_names):
            theta_unc = theta_unc + betas[(pname, cov_name)] * covariate_matrix[:, j]
    
    # Transform to constrained space
    theta = defaults["lower"] + (defaults["upper"] - defaults["lower"]) * phi_approx(theta_unc)
    sampled[pname] = numpyro.deterministic(pname, theta)
```

The per-participant loop then decodes kappa_total/kappa_share as in the single-covariate M6b model:
```python
for idx, pid in enumerate(participant_ids):
    kappa_total_i = sampled["kappa_total"][idx]
    kappa_share_i = sampled["kappa_share"][idx]
    kappa = kappa_total_i * kappa_share_i
    kappa_s = kappa_total_i * (1.0 - kappa_share_i)
    
    log_lik = wmrl_m6b_multiblock_likelihood_stacked(
        stimuli_stacked=pdata["stimuli_stacked"],
        actions_stacked=pdata["actions_stacked"],
        rewards_stacked=pdata["rewards_stacked"],
        set_sizes_stacked=pdata["set_sizes_stacked"],
        masks_stacked=pdata["masks_stacked"],
        alpha_pos=sampled["alpha_pos"][idx],
        alpha_neg=sampled["alpha_neg"][idx],
        phi=sampled["phi"][idx],
        rho=sampled["rho"][idx],
        capacity=sampled["capacity"][idx],
        kappa=kappa,
        kappa_s=kappa_s,
        epsilon=sampled["epsilon"][idx],
        num_stimuli=num_stimuli, num_actions=num_actions,
        q_init=q_init, wm_init=wm_init,
        return_pointwise=False,
    )
    numpyro.factor(f"obs_p{pid}", log_lik)
```
