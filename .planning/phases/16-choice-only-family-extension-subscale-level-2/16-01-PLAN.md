```python
from __future__ import annotations

import numpy as np
import pandas as pd


def build_level2_design_matrix(
    metrics_df: pd.DataFrame,
    participant_ids: list,
    include_lec_subcategories: bool = False,
) -> tuple[np.ndarray, list[str]]:
    """Build the standardized Level-2 design matrix for hierarchical regression.

    Parameters
    ----------
    metrics_df : pd.DataFrame
        DataFrame with columns: sona_id, ies_total, ies_intrusion,
        ies_avoidance, ies_hyperarousal, less_total_events.
    participant_ids : list
        Sorted participant IDs (must match the order used by
        prepare_stacked_participant_data).
    include_lec_subcategories : bool
        If True, include LEC-5 physical/sexual/accident columns.
        Currently not available in the data; raises ValueError if True
        and columns are missing.

    Returns
    -------
    X : np.ndarray
        Shape (n_participants, n_covariates). All columns are z-scored.
    covariate_names : list[str]
        Column names matching the columns of X. Used for naming beta_*
        sites in the NumPyro model.
    """
```

Implementation details:
1. Align `metrics_df` to `participant_ids` via `.set_index("sona_id").reindex(participant_ids)`.
2. Extract raw IES-R subscales: `ies_intrusion`, `ies_avoidance`, `ies_hyperarousal`.
3. Gram-Schmidt residualize each subscale against `ies_total`:
   `resid = subscale - np.dot(subscale, ies_total) / np.dot(ies_total, ies_total) * ies_total`
   (both arrays should be centered before projection; z-score happens after residualization).
4. Build column list: `["lec_total", "iesr_total", "iesr_intr_resid", "iesr_avd_resid", "iesr_hyp_resid"]`.
5. Z-score each column: `(col - mean) / (std + 1e-8)`.
6. Return `(X, covariate_names)`.

Also add a helper function `run_collinearity_audit()` that:
- Computes raw correlation matrix of `[ies_intrusion, ies_avoidance, ies_hyperarousal]`
- Computes condition number of raw subscale matrix
- Computes condition number of residualized subscale matrix
- Returns a dict with all diagnostics

Use `from __future__ import annotations` at top. NumPy-style docstrings. Type hints with `list[str]`, `np.ndarray`.
