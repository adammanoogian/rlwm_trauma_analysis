       ```
       sigma_K_pr ~ HalfNormal(0.2)

       ```
       Where `Phi_approx` is the standard normal CDF (`jax.scipy.stats.norm.cdf`). Explain that this maps the unconstrained real line to [2, 6] with the prior centered at 4.0 (midpoint).

    4. **parameterization_version** — The version string for all v4.0 fits using this convention is `"v4.0-K[2,6]-phiapprox"`. This column appears in all Bayesian CSV outputs and is validated on load.

       - Senta, Bishop, Collins 2025 PLOS Comp Biol (continuous [2, 6], fmincon 20 starts)

    6. **Why Lower Bound = 2** —
       - Senta 2025 convention (the project's reference paper).
       - Scientific interpretation: K < 2 means "effectively no WM" since the smallest set size in any Collins-paradigm task is ns=2. At K=2, WM weight at ns=2 is `rho * min(1, 2/2) = rho` (full reliance). At K=1, WM weight at ns=2 is `rho * 0.5` (half reliance) — but this is confounded with rho itself.
       - The v3.0 MLE pipeline used K in [1, 7]; this IS a breaking change. Phase 14 (K-02, K-03) refits all models with the new bounds.

    7. **Why Upper Bound = 6** —
       - Task max set size is ns=6. K > 6 is structurally indistinguishable from K=6 because `min(1, K/ns)` saturates at 1 for all observed ns values.
       - Senta 2025 uses the same upper bound.

    8. **BIC Rejection Rationale** — Cite Senta 2025 p.22: BIC over-penalizes RLWM models (model recovery confusion matrix confirms). Primary Bayesian criterion in v4.0 is WAIC/LOO. BIC retained in CSV for v3.0 MLE back-compatibility only.
