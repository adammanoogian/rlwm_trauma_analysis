Add function `run_parameter_recovery(model: str, n_subjects: int, n_datasets: int, seed: int, use_gpu: bool = False, verbose: bool = True) -> pd.DataFrame`:
- For each dataset (outer loop with tqdm):
  - Sample n_subjects parameter sets
  - For each subject (inner loop with tqdm, leave=False):
    - Generate synthetic data
    - Prepare data using prepare_participant_data from fit_mle.py
    - Fit via fit_participant_mle() with n_starts=50
    - Store true params, recovered params, nll, converged status
- Return DataFrame with columns: dataset, subject, true_{param}, recovered_{param}, nll, converged

Add function `compute_recovery_metrics(results_df: pd.DataFrame, model: str) -> pd.DataFrame`:
- Get parameter names for model (QLEARNING_PARAMS, WMRL_PARAMS, WMRL_M3_PARAMS from mle_utils)
- For each parameter:
  - Extract true_{param} and recovered_{param} columns
  - Compute Pearson r using scipy.stats.pearsonr
  - Compute RMSE: sqrt(mean((true - recovered)^2))
  - Compute bias: mean(recovered - true)
  - Determine pass/fail: r >= 0.80
- Return DataFrame with columns: parameter, pearson_r, p_value, rmse, bias, pass_fail
