1. Add `wmrl_m4_block_likelihood(stimuli, actions, rewards, set_sizes, rts, alpha_pos, alpha_neg, phi, rho, capacity, kappa, v_scale, A, b, t0, num_stimuli, num_actions, q_init, wm_init, mask)`:

   NOTE: The function signature takes DECODED `b` (not delta). The caller computes `b = A + delta`.

   **Carry**: Same 5-element carry as M3: `(Q, WM, WM_baseline, log_lik_accum, last_action)`
   ```python
   Q_init = jnp.full((num_stimuli, num_actions), q_init, dtype=jnp.float64)
   WM_init_mat = jnp.full((num_stimuli, num_actions), wm_init, dtype=jnp.float64)
   WM_0 = jnp.full((num_stimuli, num_actions), wm_init, dtype=jnp.float64)
   init_carry = (Q_init, WM_init_mat, WM_0, 0.0, -1)
   ```

   **Inputs to scan**: `(stimuli, actions, rewards, set_sizes, rts, mask)` -- note `rts` is new (float64, in seconds).

   **Step function body** -- the learning and policy computation is IDENTICAL to M3 up through the hybrid policy + perseveration. The key difference is the log-likelihood computation:

   a. WM decay: `WM_decayed = (1 - phi) * WM_table + phi * WM_baseline` (same as M3)
   b. Compute RL probs via softmax with beta=50 (same as M3)
   c. Compute WM probs via softmax with beta=50 (same as M3)
   d. Compute omega = rho * min(1, capacity/set_size) (same as M3)
   e. Compute hybrid_probs = omega * wm_probs + (1-omega) * rl_probs (same as M3)
   f. Apply perseveration kernel (same as M3):
      ```python
      has_prev = last_action >= 0
      Ck = jnp.eye(num_actions, dtype=jnp.float64)[jnp.maximum(last_action, 0)]
      eff_kappa = jnp.where(has_prev, kappa, 0.0)
      pi_hybrid = (1.0 - eff_kappa) * hybrid_probs + eff_kappa * Ck
      ```
   g. **M4 DIFFERENCE -- NO epsilon noise.** The hybrid policy feeds directly into drift rates:
      ```python
      # Drift rates from hybrid policy (McDougle & Collins 2021)
      v_all = v_scale * pi_hybrid  # shape: (num_actions,)

      # Adjusted time: RT - t0 (clamped for safety)
      t_star = jnp.maximum(rt - t0, 1e-6)

      # Joint log-likelihood via LBA
      log_prob = lba_joint_log_lik(t_star, action, b, A, v_all, FIXED_S)
      ```
   h. Mask: `log_prob_masked = jnp.where(valid, log_prob, 0.0)`
   i. Update Q and WM (identical to M3)
   j. Update last_action (identical to M3)
   k. Return updated carry and log_prob_masked

   Use `lax.scan` over the trial dimension.
   Return: `-final_carry[3]` (negative accumulated log-likelihood = NLL)
