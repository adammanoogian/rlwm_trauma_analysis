```python
def rfx_bms(
    log_evidence: np.ndarray,
    alpha0: float = 1.0,
    max_iter: int = 1000,
    tol: float = 1e-4,
    n_xp_samples: int = 1_000_000,
    seed: int = 42,
) -> dict[str, np.ndarray | float]:
    """Random-effects Bayesian Model Selection with PXP.

    Parameters
    ----------
    log_evidence : np.ndarray, shape (n_subjects, n_models)
        Per-participant log model evidence. For step 21.5, use
        participant-summed LOO log-likelihoods from
        ``arviz.loo(idata, pointwise=True).loo_i`` reshaped per participant.
    alpha0 : float
        Dirichlet prior concentration (uniform).
    max_iter : int
        Max VB E-step iterations.
    tol : float
        Dirichlet-parameter L_inf convergence tolerance.
    n_xp_samples : int
        Monte-Carlo samples for exceedance probability.
    seed : int
        RNG seed for Dirichlet sampling.

    Returns
    -------
    dict
        alpha : (n_models,) Dirichlet posterior parameters
        r     : (n_models,) expected model frequencies
        xp    : (n_models,) exceedance probabilities
        bor   : float, Bayesian Omnibus Risk (Rigoux 2014)
        pxp   : (n_models,) protected exceedance probabilities
    """
```

**Internal helpers (private, double-underscore not required, just `_` prefix):**

1. `_vb_dirichlet_update(log_evidence, alpha0) -> (alpha, g)`:
   - Loop up to `max_iter`.
   - At each step: `log_u = log_evidence + psi(alpha) - psi(alpha.sum())`, stabilize via row-max subtraction, exponentiate, row-normalize to get `g` (n_subjects × n_models posterior responsibilities).
   - `beta = g.sum(axis=0)`, `alpha_new = alpha0_vec + beta`.
   - Break on `np.max(np.abs(alpha_new - alpha)) < tol`.
   - Import `from scipy.special import psi` (digamma) and `from scipy.special import gammaln`.

2. `_vb_free_energy(log_evidence, alpha, alpha0_vec, g) -> float`:
   - Implements the ELBO: `F = sum_n log(sum_k g_nk * exp(lme_nk - log g_nk)) + log_dirichlet_kl(alpha, alpha0_vec)`.
   - Concretely (numerically stable form):
     ```
     F = sum over n of logsumexp(log_evidence[n] + psi(alpha) - psi(alpha.sum()))
         - sum_k (alpha_k - alpha0_vec_k) * (psi(alpha_k) - psi(alpha.sum()))
         + gammaln(alpha.sum()) - gammaln(alpha0_vec.sum())
         - sum_k (gammaln(alpha_k) - gammaln(alpha0_vec_k))
     ```
   - Use `scipy.special.logsumexp`.
   - Include unit-test-friendly numerical-stability safeguards.

3. `_exceedance_probability(alpha, n_samples, rng) -> np.ndarray`:
   - `samples = rng.dirichlet(alpha, size=n_samples)` shape `(n_samples, K)`.
   - `winners = samples.argmax(axis=1)`.
   - `return np.bincount(winners, minlength=K) / n_samples`.

4. `_bor(log_evidence, alpha, alpha0_vec) -> float`:
   - `alpha_null = np.ones_like(alpha) * (alpha0_vec.sum() + n_subjects / len(alpha))` — null-hypothesis posterior with equal frequencies.
   - Compute `F0 = _vb_free_energy(log_evidence, alpha_null, alpha0_vec, g_null)` where `g_null` is derived from the null case (uniform responsibilities).
   - Compute `F1 = _vb_free_energy(log_evidence, alpha, alpha0_vec, g)`.
   - Return `1.0 / (1.0 + np.exp(F1 - F0))`.

5. `rfx_bms` (public): call helpers, assemble return dict. Implement PXP: `pxp = (1.0 - bor) * xp + bor / K`.

**Input shape convention:** `log_evidence` strictly 2-D `(n_subjects, n_models)`. Validate with an explicit `ValueError("log_evidence must be 2-D (n_subjects, n_models), got shape ...")` check.

**No side effects at import time.** Do not print from module body.
