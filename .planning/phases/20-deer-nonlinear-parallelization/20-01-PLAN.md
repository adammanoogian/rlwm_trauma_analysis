**In `scripts/fitting/jax_likelihoods.py`**, add two functions near the existing `associative_scan_q_update` and `associative_scan_wm_update` helpers (around line ~450, after the scan helpers section):

1. `precompute_last_action_global(actions, mask)`:
   - Input: `actions` shape (T,) int, `mask` shape (T,) float
   - Output: shape (T,) int32 array where `result[t]` is the last valid action before trial t
   - `result[0] = -1` (sentinel: no previous action)
   - For valid trials (mask > 0): `result[t] = actions[t-1]`
   - For invalid/padded trials: propagate from last valid trial (use `lax.scan` for correctness with masks)
   - This is parameter-independent: run once before MCMC

2. `precompute_last_actions_per_stimulus(stimuli, actions, mask, num_stimuli=6)`:
   - Input: `stimuli` shape (T,) int, `actions` shape (T,) int, `mask` shape (T,) float, `num_stimuli` int
   - Output: shape (T,) int32 array where `result[t]` is the last action taken for `stimulus[t]`, considering trials 0..t-1
   - Uses `lax.scan` with carry `last_actions` shape (num_stimuli,) initialized to -1
   - At each trial: output `last_actions[stimulus]`, then update `last_actions[stimulus] = jnp.where(valid, action, last_actions[stimulus])`
   - This is parameter-independent: run once before MCMC

Both functions must include NumPy-style docstrings with `from __future__ import annotations` already at top of file.
