  <action>
Create `scripts/utils/plotting_utils.py` with two main functions:

1. **`get_color_palette(data_df, color_by, custom_colors=None)`**
   - Takes a DataFrame, a column name to color by, and optional custom color dict
   - Validates that `color_by` column exists in DataFrame; raises `ValueError` with available columns if not
   - Gets sorted unique categories (dropping NaN values)
   - If `custom_colors` provided, uses those (falling back to '#808080' for unmapped categories)
   - Otherwise uses `seaborn.color_palette('tab10', n_colors=N)` for up to 10 categories
   - For >10 categories, uses `matplotlib.cm.get_cmap('Set3', N)`
   - Returns dict mapping `{category: hex_color_string}`

2. **`add_colored_scatter(ax, x, y, data_df, color_by, palette, alpha=0.7, s=50, show_legend=True, **kwargs)`**
   - Plots scatter points grouped by `color_by` column
   - `x` and `y` can be column name strings OR numpy arrays
   - Iterates through sorted categories, plotting each group with its palette color
   - Adds legend with `ncol=1` for <=5 categories, `ncol=2` for 6-10, `ncol=3` for 11-15, no legend for >15
   - Returns list of PathCollection handles

3. **Module-level constant: `TRAUMA_GROUP_COLORS`**
   - Dict matching Script 15's existing GROUP_COLORS:
     ```python
     TRAUMA_GROUP_COLORS = {

         'Trauma-No Impact': '#F18F01',
         'Trauma-Ongoing Impact': '#D62246',
         'Low Exposure-High Symptoms': '#6C757D',
     }
     ```
   - This allows scripts to pass `custom_colors=TRAUMA_GROUP_COLORS` when `color_by='hypothesis_group'`

Include proper docstrings and type hints. Import matplotlib.pyplot, matplotlib.colors, seaborn, numpy. Do NOT import anything project-specific (this is a generic utility).
