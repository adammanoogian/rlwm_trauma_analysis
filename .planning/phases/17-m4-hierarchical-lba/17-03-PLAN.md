**Test 2: `test_log_delta_recovery` (M4H-02)** -- mark `@pytest.mark.slow`
This test verifies the non-centered log(b-A) parameterization can recover known ground-truth values.

Strategy:
- Generate synthetic M4 data at KNOWN ground-truth parameters for N=10 participants
- Use `wmrl_m4_multiblock_likelihood_stacked` to generate synthetic choice+RT data (forward simulation)
- OR (simpler): use a small-N fit and check that the posterior includes the generating values

Simpler approach (recommended for tractability):
- Create synthetic data with 10 participants, 3 blocks, 12 trials each
- Fit with `num_warmup=200, num_samples=300, num_chains=2` (small but enough for recovery check)
- Extract posterior means for `log_delta_mu_pr`, `A`, and `delta`
- Check that the posterior mean of `delta` is positive (basic sanity: b > A enforced)
- Check that individual `A` values are all positive
- Check that posterior samples of `delta` are all positive (exp of anything is positive)
- If possible, check group-level mean recovery against prior expectations (log_delta_mu_pr ~ Normal(0,1), so exp(0)=1.0 is the prior mean -- posterior should be in a reasonable range)

The 15% relative error threshold from M4H-02 is a cluster-scale criterion (N=154, full MCMC). For this integration test with N=10 and short chains, use relaxed checks:
- `delta > 0` for all samples (structural correctness)
- `A > 0` for all samples (structural correctness)  
- `b = A + delta > A` for all samples (the critical property)
- Posterior mean of `log_delta_mu_pr` is finite and not at prior boundary

```python
@pytest.mark.slow
def test_log_delta_recovery():
    """M4H-02: Non-centered log(b-A) produces valid delta>0 and b>A."""
    df = _make_synthetic_m4_data(n_participants=10, n_blocks=3, n_trials_per_block=12, seed=42)
    pdata = prepare_stacked_participant_data_m4(df)
    
    nuts = NUTS(wmrl_m4_hierarchical_model, target_accept_prob=0.80, max_tree_depth=8)
    mcmc = MCMC(nuts, num_warmup=200, num_samples=300, num_chains=2,
                chain_method='vectorized', progress_bar=False)
    rng_key = jax.random.PRNGKey(42)
    mcmc.run(rng_key, participant_data_stacked=pdata)
    
    samples = mcmc.get_samples()
    
    # delta must be strictly positive (exp transform guarantees this)
    delta_samples = samples["delta"]
    assert jnp.all(delta_samples > 0), "delta must be > 0 (log-scale parameterization)"
    
    # A must be strictly positive
    A_samples = samples["A"]
    assert jnp.all(A_samples > 0), "A must be > 0 (log-scale parameterization)"
    
    # b = A + delta must be > A (structural property of M4H-02)
    b_samples = A_samples + delta_samples
    assert jnp.all(b_samples > A_samples), "b must be > A (enforced by delta > 0)"
    
    # Group-level log_delta_mu_pr should be finite
    log_delta_mu = samples["log_delta_mu_pr"]
    assert jnp.all(jnp.isfinite(log_delta_mu)), "log_delta_mu_pr has non-finite values"
    
    # All 10 params should be present
    expected_params = ["alpha_pos", "alpha_neg", "phi", "rho", "capacity",
                       "kappa", "v_scale", "A", "delta", "t0"]
    for p in expected_params:
        assert p in samples, f"Missing parameter: {p}"
        assert jnp.all(jnp.isfinite(samples[p])), f"{p} has non-finite values"
```

**Test 3: `test_checkpoint_resume` (M4H-04)** -- mark `@pytest.mark.slow`

This test verifies that warmup state can be pickled and used to resume sampling.

Strategy:
- Run warmup only (`mcmc.warmup(...)`)
- Pickle `jax.device_get(mcmc.post_warmup_state)` to a temp file
- Create a NEW MCMC object
- Load the pickle, set `mcmc.post_warmup_state = state`
- Run sampling (`mcmc.run(state.rng_key, ...)`)
- Verify: posterior samples exist, are finite, have expected shape

This does NOT kill a process (that's fragile in CI). Instead it tests the checkpoint API directly, which is what the script uses.

```python
@pytest.mark.slow
def test_checkpoint_resume():
    """M4H-04: Warmup state can be pickled and used to resume sampling."""
    df = _make_synthetic_m4_data(n_participants=3, n_blocks=2, n_trials_per_block=8, seed=99)
    pdata = prepare_stacked_participant_data_m4(df)
    model_kwargs = {"participant_data_stacked": pdata}
    
    # Phase 1: warmup only
    nuts1 = NUTS(wmrl_m4_hierarchical_model, target_accept_prob=0.80, max_tree_depth=8)
    mcmc1 = MCMC(nuts1, num_warmup=50, num_samples=50, num_chains=1,
                 chain_method='vectorized', progress_bar=False)
    rng_key = jax.random.PRNGKey(42)
    mcmc1.warmup(rng_key, **model_kwargs)
    
    # Save warmup state
    with tempfile.NamedTemporaryFile(suffix=".pkl", delete=False) as f:
        warmup_path = Path(f.name)
        warmup_state_cpu = jax.device_get(mcmc1.post_warmup_state)
        pickle.dump(warmup_state_cpu, f)
    
    # Phase 2: resume from checkpoint
    try:
        nuts2 = NUTS(wmrl_m4_hierarchical_model, target_accept_prob=0.80, max_tree_depth=8)
        mcmc2 = MCMC(nuts2, num_warmup=50, num_samples=50, num_chains=1,
                     chain_method='vectorized', progress_bar=False)
        
        with open(warmup_path, "rb") as f:
            loaded_state = pickle.load(f)
        mcmc2.post_warmup_state = loaded_state
        mcmc2.run(loaded_state.rng_key, **model_kwargs)
        
        # Verify: samples exist and are finite
        samples = mcmc2.get_samples()
        assert len(samples) > 0, "No samples after resume"
        for param_name, vals in samples.items():
            assert jnp.all(jnp.isfinite(vals)), f"Non-finite samples for {param_name} after resume"
    finally:
        warmup_path.unlink(missing_ok=True)
```
