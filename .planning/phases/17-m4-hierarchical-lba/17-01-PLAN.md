**Function 2: `wmrl_m4_hierarchical_model`**

Follow the M6b hierarchical model (line 1627) as the structural template. Key differences:

Signature:
```python
def wmrl_m4_hierarchical_model(
    participant_data_stacked: dict,
    num_stimuli: int = 6,
    num_actions: int = 3,
    q_init: float = 0.5,
    wm_init: float = 1.0 / 3.0,
) -> None:
```

NO covariate_lec parameter (no Level-2 regression for M4). NO epsilon parameter (M4 has 10 params, not 11).

Parameter sampling:
1. Six RLWM params via `sample_bounded_param` loop: `["alpha_pos", "alpha_neg", "phi", "rho", "capacity", "kappa"]`. Import `sample_bounded_param`, `PARAM_PRIOR_DEFAULTS`, `phi_approx` from `numpyro_helpers` at top of function (lazy import pattern matching M6b).
2. v_scale: log-normal non-centered. `log_v_mu_pr ~ Normal(log(3.0), 0.5)`, `log_v_sigma_pr ~ HalfNormal(0.2)`, `log_v_z ~ Normal(0,1).expand([n_participants])`. Deterministic site: `v_scale = exp(log_v_mu_pr + log_v_sigma_pr * log_v_z)`.
3. A: log-normal non-centered. `log_A_mu_pr ~ Normal(log(0.3), 0.5)`, `log_A_sigma_pr ~ HalfNormal(0.2)`, `log_A_z ~ Normal(0,1).expand([n_participants])`. Deterministic site: `A = exp(log_A_mu_pr + log_A_sigma_pr * log_A_z)`.
4. delta (M4H-02): log-normal non-centered. `log_delta_mu_pr ~ Normal(0.0, 1.0)`, `log_delta_sigma_pr ~ HalfNormal(0.2)`, `log_delta_z ~ Normal(0,1).expand([n_participants])`. Deterministic site: `delta = exp(log_delta_mu_pr + log_delta_sigma_pr * log_delta_z)`.
5. t0: bounded probit in [0.05, 0.3]. `t0_mu_pr ~ Normal(0.0, 1.0)`, `t0_sigma_pr ~ HalfNormal(0.2)`, `t0_z ~ Normal(0,1).expand([n_participants])`. Deterministic site: `t0 = 0.05 + 0.25 * phi_approx(t0_mu_pr + t0_sigma_pr * t0_z)`.

Likelihood loop (Python for-loop over participants, matching M6b pattern):
