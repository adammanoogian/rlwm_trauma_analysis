<objective>
Implement the M6a stimulus-specific perseveration model: JAX likelihood functions with per-stimulus last_actions carry array replacing the global last_action scalar, MLE parameter registration, and full fitting pipeline integration.

Purpose: M6a replaces M3's global perseveration kernel (which applies to any stimulus after any action was taken in the block) with a stimulus-specific kernel that only applies to a stimulus after IT has been acted upon. This tests whether perseveration is stimulus-bound rather than globally motor-based. The parameter kappa_s replaces kappa with identical bounds [0,1] -- same 7 parameters as M3.

Output: Three files modified such that `python scripts/12_fit_mle.py --model wmrl_m6a` runs end-to-end with per-stimulus last_action tracking in the lax.scan carry.
</objective>

     g. Update per-stimulus slot (UNCONDITIONALLY on valid trial, regardless of whether kernel was applied):
